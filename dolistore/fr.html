Fonctionnalités principales :

  • Processus en trois étapes intuitif et fluide
  • Téléversement de fichiers multiples avec détection des doublons
  • Extraction automatique des données via l’API UpInvoice
  • Système avancé de recherche et de création de fournisseurs
  • Validation des données des factures avec calculs automatiques
  • Création des factures fournisseurs avec ou sans validation
  • Interface moderne et adaptée à Dolibarr

Nouveauté – Collecte automatique depuis la messagerie :

Le module peut désormais collecter les factures reçues directement depuis une boîte mail, sans téléversement manuel. Grâce à l’intégration avec le module EmailCollector de Dolibarr, les pièces jointes (PDF ou images) arrivant dans votre boîte sont importées automatiquement dans la liste des fichiers en attente. Depuis la page de configuration, vous pouvez créer un collecteur préconfiguré en un seul clic et activer le traitement automatique par IA afin qu’une tâche planifiée (cron) extraie les données des factures entrantes sans aucune intervention manuelle.

De plus, des règles d’import (par expéditeur, sujet ou motif de nom de fichier) et une liste noire pour exclure les expéditeurs indésirables vous permettent de n’importer que les messages et pièces jointes qui sont réellement des factures. Un onglet de boîte mail en direct vous permet de consulter les emails reçus et d’importer des pièces jointes manuellement quand vous le souhaitez.

Pré-requis :

  • Version minimale de Dolibarr : 14.0.0
  • Version maximale de Dolibarr : 23
  • Clé API UpInvoice (disponible sur https://upinvoice.eu)
  • Modules Fournisseurs et Factures Fournisseurs activés

Configuration :

  1. Installer le module via Dolistore ou manuellement
  2. Aller dans Configuration > Modules et activer le module UpInvoice
  3. Dans la configuration du module, entrer l’URL de l’API et la clé API d’UpInvoice
  4. Prêt à l’emploi ! Accès depuis le menu principal

Le module s’intègre parfaitement dans le flux de travail de Dolibarr, en joignant automatiquement les documents originaux aux factures créées. Pour un fonctionnement optimal, il est recommandé d’activer le module de Gestion Électronique de Documents (GED).

Avertissement : Un abonnement au service UpInvoice.eu est requis pour connecter son API avec ce module. Veuillez consulter la page des tarifs sur https://upinvoice.eu/prices afin de choisir l’abonnement adapté à vos besoins.  
Remarque : Vous pouvez traiter gratuitement un nombre limité de factures pour tester le service et constater le gain de temps qu’il vous apporte.
