Características principales:

  • Proceso en tres pasos intuitivo y fluido
  • Subida de múltiples archivos con detección de duplicados
  • Extracción automática de datos mediante API de UpInvoice
  • Sistema avanzado de búsqueda y creación de proveedores
  • Validación de datos de facturas con cálculos automáticos
  • Creación de facturas de proveedor con o sin validación
  • Interfaz moderna y adaptada a Dolibarr

Novedad – Captura automática desde el correo electrónico:

El módulo ahora puede recoger las facturas recibidas directamente desde un buzón de correo, sin necesidad de subirlas manualmente. Gracias a la integración con el módulo EmailCollector de Dolibarr, los adjuntos (PDF o imágenes) que llegan a su buzón se importan automáticamente a la lista de archivos pendientes. Desde la página de configuración puede crear el recolector preconfigurado con un solo clic, y activar el procesado automático con IA para que un trabajo programado (cron) extraiga los datos de las facturas entrantes sin ninguna intervención manual.

Además, dispone de reglas de importación (por remitente, asunto o patrón de nombre de archivo) y de una lista negra para excluir remitentes no deseados, de modo que solo se importen los mensajes y adjuntos que realmente son facturas. Una pestaña de buzón en vivo le permite ver los correos recibidos e importar adjuntos de forma manual cuando lo prefiera.

Requisitos:

  • Versión mínima de Dolibarr: 14.0.0
  • Versión máxima de Dolibarr: 23
  • Clave API de UpInvoice (disponible en https://upinvoice.eu)
  • Módulo de Proveedores y Facturas de Proveedores activado

Configuración:

  1. Instalar el módulo desde Dolistore o mediante carga manual
  2. Acceder a Configuración > Módulos y activar el módulo UpInvoice
  3. En la configuración del módulo, introducir la URL de la API y la clave API de UpInvoice
  4. ¡Listo para usar! Accede desde el menú principal

El módulo se integra perfectamente con el flujo de trabajo de Dolibarr, permitiendo adjuntar automáticamente los documentos originales a las facturas creadas. Para un funcionamiento óptimo, se recomienda tener activado el módulo de Gestión Electrónica de Documentos (GED).

Nota: Podrá procesar algunas facturas de forma gratuita para probar el servicio y descubrir el ahorro de tiempo que le ofrece.
